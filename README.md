# Climate Change Modeling — NLP & Sentiment Analysis

A machine learning project analyzing public opinion on climate change using 500+ user comments
from NASA's Facebook page (2020–2023). Built as part of a Data Science internship at Unified Mentor.

## What This Project Does
- Sentiment Analysis using VADER & TextBlob
- Topic Modeling using LDA (5 key themes discovered)
- ML Classification (Logistic Regression, SVM, Random Forest & more)
- EDA with temporal trends, engagement analysis & word clouds

##  Tech Stack
Python · Jupyter / Google Colab · scikit-learn · NLTK · TextBlob · VaderSentiment · Gensim · Matplotlib · Seaborn

## Dataset
500+ comments scraped from [NASA Climate Change Facebook Page](https://web.facebook.com/NASAClimateChange/)
covering posts from 2020–2023. Features: date, text, likes, comments count.
Profile names are SHA-256 hashed for privacy.

##  How to Run
1. Clone the repo
2. Open `Climate_Change_Modeling_Colab.ipynb` in Google Colab
3. Upload `climate_nasa.csv` when prompted
4. Run all cells
